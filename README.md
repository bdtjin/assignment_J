# 개인과제_박진
### 과제 설명

### 1. 첫번째 문제

#### [문제: 제시된 예시 점수에 해당하는 등급 출력하기]
- 예시 점수: 84점
- 등급 기준 
  - A학점 - 90~100점
  - B학점 - 80~89점
  - C학점 - 80점 이하

해당 문제는 먼저 예시 점수를 조건에 넣어준 뒤에 `조건문(if/else if)`를 통해 등급을 판정할 코드를 넣어서 풀 수 있었습니다. 판정할 코드 안에는 `비교 연산자`를 넣어서 학점(값)의 조건 처리를 구현할 수 있었습니다. 

---
### 2. 두번째 문제

#### [문제: 구매자가 장바구니에 담은 금액을 최종 출력하기]
- 조건: 총 금액이 20,000원이상일 경우, 10% 할인 적용
- 구매자의 장바구니 품목: 티셔츠2, 바지2, 모자1
- 가게에서 판매하고 있는 상품과 각각의 가격
  - 티셔츠: 10000원
  - 바지: 8000원
  - 모자: 4000원

해당 문제는 반복문(for)과 조건문(if)을 활용하여 최종 금액을 산출해야되는 시스템으로 조건 처리를 구현한 후에 최종 금액을 출력할 수 있습니다.

핵심은 반복적인 합산과 조건부 할인 로직을 이해하는 과정이었다. 과정 중에 자주 버벅거렸던 부분은 반복문(for)문을 사용해서 list를 순회하며 각 상품 이름에 해당하는 가격을 Map에서 조회하고, 이를 누적변수에 합산하는 과정에 있었다. 또한 제대로 if문에서 제대로 작성했는데 자꾸 오류가 뜨는 부분에 있어서 다시한번 확인 시에는 정수형으로 확정지어주는 부분도 기본 개념을 다시한번 짚을 수 있는 문제였다.
